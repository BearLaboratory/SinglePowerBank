# SinglePowerBank

BLab大熊实验室单节锂电池移动电源，或者也可以叫做锂电池充电。

- 充放电IC使用ETA9638
- USB-TYPEC插头
- 可为普通18650锂电池充放电
- 也可使用使用普通锂电池

------------------------------

## PCB尺寸

- 长度：95mm
- 宽度：22mm
